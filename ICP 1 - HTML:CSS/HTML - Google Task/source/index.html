Let's get started!

Before creating your first repository on github, please make sure you have created an account. Click here to get started

1. Initiate Repository Creation
  • Navigate to the Repositories tab in your github profile and select the new button to initiate creation of repository as depicted below.
  • Note: If you are not redirected to a new page double check that you are clicking the correct button.

2. Define Repository Information

  1. Fill in Repository name with your desired name
  2. Input an optional description for your repository.
  3. Set your repository with the desired publicity status.
  4. Select option for README.md if you would prefer.
  5. If a .gitignore is necessary for your project, select it as well.
  6. Select Create Repository to initate Repo creation!

3. Utilize your Repository

Congratulations! You have officially created your first Github repository. Let's enjoy the fruits of our labor.

  • You can now clone your remote repository locally by utilizing the .git link or using Github for Desktop
  • You may also choose to import code from an existing repository!
